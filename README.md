# Java Code Reading Practice

## 概要
本リポジトリは、Javaのコードリーディング練習のためのサンプルコードです。

コード本体は生成AIを用いて作成し、
コード内のコメントは自分自身で内容を理解しながら記述しました。

## 目的
・既存コードを読む力を養う  
・メソッド、配列、クラス、コンストラクタの理解を深める  
・処理の流れを正確に説明できるようになる  

## 内容

### Practice1
- 配列
- for文
- if文
- メソッドの戻り値
- 偶数の合計処理

### Practice2
- クラス定義
- フィールド
- コンストラクタ
- booleanメソッド
- オブジェクト生成とメソッド呼び出し

## 学習方法
1. コードを読む
2. 処理内容を自分の言葉でコメントとして記述する
3. 不明点を整理し理解を深める

## 補足
本リポジトリは学習目的のものであり、
商用利用や実務利用を想定したコードではありません。
